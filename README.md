# ASCEND2026 – Open-Source CubeSat Missions (TGC × AEROO)

This repo hosts ready to run CubeSat code, mission templates, and an open gallery of student projects from ASCEND 2026.

## What’s inside
- `examples/arduino` – firmware sketches for sensors/comms
- `examples/python` – data parsing, viz, alerting
- `kits/` – hardware setup & wiring diagrams
- `docs/` – mission briefs, worksheets, rubrics
- `submissions/` – student teams’ open projects

## Quick start
1. Clone or fork the repo.
2. Open an example in `examples/arduino` (Arduino IDE) or `examples/python` (Python 3.10+).
3. See `docs/setup.md` for wiring, flashing, and dashboard login.

## Contributing
- Mentors: open PRs to `examples/*` and `docs/*`.
- Students works will be added and maintained by mentors: add project under `submissions/<team-name>-<mission>/` with a README, code, and LICENSE.
See `CONTRIBUTING.md` for style and review rules.
