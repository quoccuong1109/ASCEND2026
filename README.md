# ASCEND2026 Open-Source CubeSat Missions (TGC × AEROO)

This repo hosts ready to run CubeSat code, and an open gallery of student projects from ASCEND 2026.

## What’s inside
- `Cubesat_esp32/` – Full Satellite Code Library
- `submissions/` – ASCEND2026 student teams’ open projects (will be uploaded and maintained after ASCEND2026)

## Quick start
1. Clone or fork the repo; or
2. Click on file you want to extract from repo, and `view raw` to download the `.zip` files

## Contributing
- Mentors: open PRs to main repo
- Students works will be added and maintained by mentors: add project under `submissions/<team-name>-<mission>/` with a README, code, and LICENSE.
